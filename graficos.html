Capa del Sistema Tecnologías Implementadas Justificación Técnica
Framework Web Flask (Python) Seleccionado por su minimalismo, flexibilidad y adecuación para aplicaciones de tamaño medio. Permite una configuración precisa y evita la sobrecarga de funcionalidades innecesarias.
Base de Datos MongoDB Optado por su naturaleza documental que se adapta naturalmente a la estructura semi-estructurada de los datos extraídos de portales de empleo. Facilita la evolución del esquema sin migraciones costosas.
Extracción de Datos BeautifulSoup4 + Requests Combinación madura y robusta para web scraping, con capacidad para manejar variaciones en la estructura de páginas web y mecanismos de tolerancia a fallos.
Interfaz de Usuario Jinja2 Templates + CSS/JS Proporciona un balance óptimo entre simplicidad y funcionalidad, permitiendo interfaces responsivas sin la complejidad de frameworks frontend pesados.
